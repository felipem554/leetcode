# LeetCode Solutions

Simple collection of **Java solutions** for selected [LeetCode](https://leetcode.com/) problems.
Used mainly to track my progress and test different approaches.

## 🧠 About

Each folder represents a problem. Inside you’ll find the `Solution.java` file with a working implementation — usually the cleanest or most efficient one I’ve found.

## 📂 Structure

```
/
├── two_sums/
├── two_sums_2/
├── remove_duplicates_sorted_array/
├── palindrome/
├── majority_element/
└── ...
```

## ▶️ Run

Clone the repo and compile any solution:

```bash
git clone https://github.com/felipem554/leetcode.git
cd leetcode/two_sums
javac Solution.java
java Solution
```
